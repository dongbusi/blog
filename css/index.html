      # marigin和padding的基准

      maigin和padding都是有百分比取值的，他的百分比的基准是他的父元素的宽。


       












       
       



      <div class="father">
  <div class="child"></div>
</div> 
<style>

.father {
  width: 300px;
  height: 200px;
  border: 1px dashed red;
}
.child {
  width: 150px;
  height: 100px;
  border: 1px solid #cccccc;
  margin-top: 10%; // padding 同理，不再赘述
  margin-left: 10%;
}
</style>

      # 居中

      居中是css布局中最为常见的需求，居中的方式也多种多样。这里列举一些简单且本人经常使用的。

      # 文字居中

      文字
      <div class="content">文字</div>
  
<style lang="scss">
  .content {
    width: 100px;
    height: 100px;
    text-align: center;
    line-height: 100px; // 父元素高度 （百分比会报错）
  }
</style>

      # 图片居中

      <div class="container">
  <img width="100" height="100" src="/img/css-margin.jpeg" >
  <div class="iblock"></div>
</div>

  .container {
    margin-top: 20px;

    text-align: center;

    vertical-align: middle;
  }

    display: inline-block;
    vertical-align: middle;
    height:100%;
	  width:0;
